Manualne testy adapterów

Sprawdzaj helper endpointy Elastica i GitLaba bez wychodzenia z aplikacji.

Widok służy do ręcznego odpalania adapterów diagnostycznych. Każda karta wysyła request do backendu i pokazuje sformatowaną odpowiedź JSON w polu tekstowym.

JSON response preview Elastic log search Elastic HTTP compare GitLab repo mapping GitLab source resolve

Przydatne scenariusze

  • czy `correlationId` zwraca oczekiwane logi z Elastica
  • czy podobne wywołania HTTP na tym samym path kończą się 200/400/500
  • czy hint komponentu mapuje się na właściwe repozytorium GitLaba
  • czy symbol klasy rozwiązuje się do pliku i preview kodu
